Le projet contient le layout_relative qui remplit la partie 1 du TD et remplit les fonctions suivantes :

- Ajout d'une ressource image
- Changer le fond d'écran
- Test du RelativeLayout avec un bouton, un label et une text box.

La gallery qui remplit la partie 2 du TD et remplit les fonctions suivantes :

- Création de la classe interne imageAdapter
- Test du GridView
- Affichage d'une série d'images ordonnées dans le GridView 
- Test des Toasts

Le projet démarre sur la Gallery.
Pour démarrer sur la partie 1 il faut remplacer :         setContentView(R.layout.gallery_photos);     par      setContentView(R.layout.activity_main);
